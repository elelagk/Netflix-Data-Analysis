## Overview
This project was developed for the **Data Mining Techniques** course at the National and Kapodistrian University of Athens. 

It explores and visualizes Netflix data and implements a **content-based recommendation system** using Netflix & IMDb data.

---

## Objectives
- **Data Cleaning:** Handle missing values, remove duplicates, and standardize key columns.  
- **Data Analysis:** Compare movies vs. TV shows, identify content trends, and visualize categories, countries, ratings, and genres.  
- **IMDb Integration:** Merge Netflix and IMDb datasets to highlight top-rated Netflix titles.  
- **Recommendation System:**  
  - Build a content-based recommender using *Bag-of-Words* and *TF-IDF* representations.  
  - Measure similarity with *Jaccard/Tanimoto* and *Cosine similarity*.
